# VSCode Devcontainer Template for PHP

## Abstracts

| layer       | name    | version |
| :---------- | :------ | ------: |
| OS          | alpine  |  latest |
| application | PHP     |     7.4 |
| database    | MariaDB |    10.2 |

## How to setup

Create a `.env` file, then you can run this container.

```
$ cp ./.devcontainer/.env.example ./.devcontainer/.env
```
